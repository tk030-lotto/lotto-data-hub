# lotto-data-hub

ロト・ナンバーズの過去当選結果データを自動取得し、JSON形式で配信する専用のデータハブです。

## 概要

このリポジトリは、日本の主要なロト・ナンバーズくじの最新の過去データを定期的に取得し、JSONファイルとしてGitHub Pages経由で公開します。
各種Webベースの解析・判定ツールは、このリポジトリが配信するデータを参照します。

## 配信データ一覧

* ロト6: `https://tk030-lotto.github.io/lotto-data-hub/data/loto6.json`
* ロト7: `https://tk030-lotto.github.io/lotto-data-hub/data/loto7.json`
* ミニロト: `https://tk030-lotto.github.io/lotto-data-hub/data/miniloto.json`
* ナンバーズ3: `https://tk030-lotto.github.io/lotto-data-hub/data/numbers3.json`
* ナンバーズ4: `https://tk030-lotto.github.io/lotto-data-hub/data/numbers4.json`
